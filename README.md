# 用DOCKER封装的三层Web应用

### 快速启动：

1. 确保你安装了docker

2. 登录docker hub

   ```docker
   docker login
   ```

3. 拉取远程仓库的镜像

   ```docker
   docker pull xyx138/node-mysql
   ```

4. 下载`docker-compose.yml`到本地

5. 启动项目

   ````yml
   docker-compose up --build
   ````

6. 点击访问项目
